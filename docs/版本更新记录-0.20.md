## 0.20 重点更新说明

### 1. 多渠道榜单获取

0.20 版本开始，榜单模块不再依赖单一来源，而是支持多渠道切换与回退，用于提升数据可用率与参考价值。

当前重点覆盖：

- FANZA / DMM 可用路径优先尝试
- AVfan 在线榜单作为补充来源
- 本地历史榜单导入用于补足历史缺口
- 多来源回退逻辑，尽量避免单一渠道失效后整块功能不可用

### 2. 榜单多元化

本版本为榜单模块补充了更完整的月度 / 年度读取思路，并支持本地历史榜单目录，便于后续扩展更多来源。

这意味着：

- 在线榜单可以直接刷新
- 历史榜单可以由用户手动补充到本地目录
- 软件会优先读取更匹配的本地历史记录
- 后续扩展更多榜单渠道时，底层结构已预留扩展位

### 3. 安装体验完善

0.20 对安装包流程进行了进一步优化：

- 安装向导保持全中文显示
- 支持自定义安装路径
- 选择目标路径后可自动创建目录
- 安装过程中会显示更明确的进度与步骤说明
- 安装后的 `docs` 目录会附带版本更新记录、软件说明书和开发说明书

### 4. 抓取优先级与已知问题修复

本次版本同步处理了以下稳定性问题：

- 优化榜单来源切换优先级，降低错误来源抢占结果的问题
- 修复部分缓存回退逻辑导致年份 / 月份不准确的问题
- 修复安装器文案与流程说明不完整的问题
- 优化抓取链路中的优先级判断，减少错误分流带来的异常行为

## 升级建议

- 建议从旧版本升级到 0.20 时，先备份已有输出目录与配置目录
- 如需使用榜单功能，建议开启稳定代理 / VPN，尤其是日本地区代理
- 如需补历史月榜或年榜，建议优先准备可信来源的数据后再导入本地历史目录
