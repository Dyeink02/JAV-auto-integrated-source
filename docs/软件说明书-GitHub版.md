# JAV自动化爬虫工具 软件说明书

## 📖 产品概述

**JAV自动化爬虫工具**是一款专为 Windows 平台设计的 JAV（Japanese Adult Video）内容自动化抓取与视频整理工具。它将高性能爬虫引擎、智能视频整理、AI 广告检测三大核心能力集成于一个美观的桌面应用中，让用户只需点击几个按钮即可完成从"网页抓取"到"本地归档"的完整工作流。

### 产品定位
- **一站式解决方案**：从爬取磁力链接到整理本地视频库，全流程在一个软件内完成
- **小白友好**：可视化 GUI 界面，全中文操作，无需编写代码或记忆命令
- **智能高效**：四阶段流水线并发架构、断点续爬、AI 广告检测、自动番号匹配
- **可定制**：支持代理配置、并行量调整、模型策略选择、自定义背景

### 核心模块说明

#### 1. 爬虫引擎（`src/core/`）
- **QueueManager**：四阶段流水线编排器
  - 阶段 1：索引页面抓取（1x 并发）
  - 阶段 2：详情页面解析（0.75x 并发）
  - 阶段 3：文件写入（2x 并发，高优先级）
  - 阶段 4：图像下载（0.5x 并发，低优先级）
- **RequestHandler**：带重试和代理支持的 HTTP 客户端
- **Parser**：基于 Cheerio 的 HTML 解析器，支持多站点回退策略
- **PuppeteerPool**：管理 Chromium 浏览器实例池，用于 Cloudflare 绕过
- **FilmIdentity**：番号标准化与身份识别模块

#### 2. 整理引擎（`desktop/mainServices/organizerModules/`）
- **judgePhase**：视频分类判定（待整理 / 含开头广告 / 待删除）
- **renamePhase**：按番号规则重命名视频文件
- **adLearningService**：广告样本学习与 AI 模型推理
- **organizerService**：整理流程总调度器

#### 3. 前端 UI（`desktop/renderer/`）
- **renderer.js**：主渲染进程，页面切换与状态管理
- **formController.js**：表单输入控制
- **logController.js**：日志面板渲染
- **rankingController.js**：女优榜单渲染
- **organizerController.js**：整理任务控制

## 🔧 核心功能

### 一、爬虫功能
| 功能 | 说明 |
|------|------|
| **磁力链接爬取** | 自动提取影片元数据、分类、演员信息，通过 AJAX 获取磁力链接 |
| **并发图像下载** | 海报图片并发下载，带 referer 头保护 |
| **断点续爬** | 任务状态落盘，中断后可恢复继续爬取 |
| **补抓队列** | 三段式补抓（索引页 → 快速 HTTP → Cloudflare 补抓） |
| **Cloudflare 绕过** | 基于 Puppeteer 的年龄验证绕过，调用用户本地 Chromium |
| **代理支持** | 自动检测系统代理，支持 HTTP/HTTPS/SOCKS4/SOCKS5 |
| **磁力内容校验** | 可选开关，自动识别广告包/杂文件包并跳过 |

### 二、视频整理功能
| 功能 | 说明 |
|------|------|
| **自动番号匹配** | 扫描本地视频文件夹，自动提取番号并与爬虫结果对照 |
| **智能重命名** | 命中爬虫番号的视频自动按标准格式重命名 |
| **分类分流** | 按体积+番号规则分类：待整理 / 含开头广告 / 待删除 |
| **全链路对账** | 输出遗漏番号补抓报告，方便客户快速补下 |
| **目录清理** | 整理完成后自动清理空目录，保持目录结构整洁 |
| **删除优化** | 视频移动后原文件夹直接整目录删除，大幅提升速度 |

### 三、AI 广告检测
| 功能 | 说明 |
|------|------|
| **AI 模型推理** | 内嵌 ONNX Runtime，支持三种模型策略 |
| **帧采样检测** | 视频前 15 秒密集帧采样，dHash 感知哈希比对 |
| **样本学习** | 按番号自动学习，支持导入广告/正常样本 |
| **可视化开关** | 一键启用/关闭 AI 检测，选择模型策略 |
| **回灌学习** | 误判样本可回灌学习库，识别效果持续提升 |

### 四、女优排行榜
| 功能 | 说明 |
|------|------|
| **多渠道榜单** | FANZA/DMM 月榜年榜、AVfan 在线榜、本地历史榜 |
| **一键参数填充** | 点击榜单自动填充爬虫参数（女优目录/数量/页数） |
| **多来源回退** | 单一渠道失效时自动切换到备用来源 |

## 📋 安装与使用

### 系统要求
| 组件 | 要求 |
|------|------|
| **操作系统** | Windows 10 / Windows 11（64 位） |
| **浏览器** | Google Chrome 或 Chromium（用于 Cloudflare 绕过） |
| **内存** | 推荐 4 GB 以上 |
| **磁盘** | 安装包约 150-200 MB（含 ONNX 模型和 FFmpeg） |
| **网络** | 需要稳定代理/VPN 访问日本地区网站 |

### 安装步骤
1. 下载安装包（`.exe` 文件）
2. 双击运行安装程序
3. 选择安装路径（默认 `C:\Program Files\JAV自动化爬虫工具`）
4. 等待安装完成
5. 从桌面快捷方式或开始菜单启动

### 首次使用
1. **配置代理**（推荐）：在设置中填入你的代理服务器地址
2. **启用 Cloudflare 绕过**（推荐）：确保系统已安装 Chrome 浏览器
3. **输入爬取地址**：粘贴 JAV 网站的女优目录或列表页面 URL
4. **设置爬取数量**：输入要爬取的页面数量
5. **点击"开始爬取"**：等待任务完成

### 视频整理使用
1. **切换到"视频整理"页签**
2. **设置源目录**：选择存放下载视频的根目录
3. **设置输出目录**：选择整理后存放的位置
4. **配置广告处理方式**：选择"移入待删除"或"直接删除"
5. **启用 AI 检测**（可选）：选择模型策略
6. **点击"开始整理"**

## ⚠️ 注意事项

1. **代理要求**：爬取日本网站需要稳定的代理/VPN，推荐日本地区代理
2. **Chromium 依赖**：Cloudflare 绕过功能需要系统已安装 Google Chrome 或 Chromium 浏览器
3. **首次运行**：AI 广告检测功能首次使用时可能需要下载 FFmpeg（约 35 MB）

5. **数据安全**：所有配置和日志存储在用户本地，不上传任何服务器

```bash

git clone https://github.com/raawaa/jav-scrapy.git
```
