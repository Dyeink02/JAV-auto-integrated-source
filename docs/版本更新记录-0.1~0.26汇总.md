## 详细版本更新内容

### v0.1 — 安装修复
- 修复安装流程。
- 打通基础运行链路。
- 完成最初的桌面可用化准备。

### v0.2 — 桌面 GUI 上线
- 桌面版 GUI 上线。
- 支持 Windows 直接打开使用。
- 从命令行工具正式迈向可视化桌面应用。

### v0.3 — 配置增强
- 增加地址输入、输出位置、并行量、延迟等选项。
- 优化桌面布局与表单结构。
- 提升小白用户直接使用体验。

### v0.4 — 分页校验
- 增强分页校验逻辑。
- 优化页锁定补抓。
- 完善尾页数量判断。

### v0.5 — 全面汉化
- 整体汉化桌面界面。
- 加入页数判断逻辑。
- 状态说明与主要交互统一中文化。

### v0.6 — 补爬功能
- 新增补爬功能。
- 增加弹窗提醒。
- 支持磁力 TXT 导出。
- 增加未完成番号显示。

### v0.7 — 日志修复
- 修复日志乱码问题。
- 统一日志编码与显示逻辑。
- 提升问题排查体验。

### v0.8 — 断点续爬
- 加入任务状态落盘。
- 支持断点续爬与结果二次校验。
- 提升长任务中断后的恢复能力。

### v0.9 — 注释完善
- 补充底层注释，方便后续维护与 AI 协作。
- 优化滚动区域与超长任务显示。
- 调整打包体积与资源组织方式。

### v0.10 — 解析容错
- 优化解析容错。
- 调整终止流程。
- 强化补抓去重与缺失修复逻辑。

### v0.11 — 备用网址
- 补充备用网址。
- 修复一批已知问题。
- 改善日志与页面卡顿相关问题。

### v0.12 — 大任务稳态
- 大任务场景下增强稳定性。
- 增加磁力文档直接打开入口。
- 优化长时间运行过程中的操作体验。

### v0.13 — 批量写盘
- 补查分页缺口。
- 修正真实并发执行逻辑。
- 改为 HTTP 优先并引入批量写盘，减少 IO 开销。

### v0.14 — 三段队列
- 修正完成判定逻辑。
- 启用索引页 / 快速 HTTP / Cloudflare 补抓三段队列。
- 提升大批量任务下的补抓效率。

### v0.15 — 速度提升
- 大幅提升抓取速度。
- 增强抓取目标准确率与动态任务栏表现。
- 优化大任务抓取过程中的稳定性与速度平衡。

### v0.16 — 精简输出
- 收口输出文件，仅保留核心结果文件与日志。
- 强化完成判定，自动生成未完成番号文件。
- 升级失败重试策略与分页可信度评分。

### v0.17 — 界面焕新
- 焕新桌面布局与状态面板，整体风格更年轻化。
- 补充版本历史展示，并修复 `limit` 模式下的超量入队问题。
- 强化中文显示、大任务稳态与缺口判断。

### v0.18 — 女优排行榜
- 新增 FANZA 女优参考榜模块，支持月榜与年榜查看。
- 支持榜单点击后一键填充爬虫参数：自动定位真实女优目录、填入有磁力数量、每页条数与预计页数。
- 优化榜单区域界面：收窄筛选控件，约束超长网址显示，交互提示更直观。

### v0.19 — 代码解耦
- 继续推进桌面端与抓取核心的耦合拆分，降低后续改动的连带风险。
- 优化抓包速率与恢复链路，重点处理大任务下的 UI 卡顿与状态堆积。
- 新增背景自定义模块，支持用户自行更换桌面背景图片。
- 修复 base-url 快捷地址点击、日志面板与状态面板在高频更新场景下的卡顿问题。

### v0.20 — 多渠道榜单
- **多渠道榜单获取**：榜单模块不再依赖单一来源，支持多渠道切换与回退（FANZA/DMM、AVfan、本地历史榜单）。
- **榜单多元化**：补充更完整的月度 / 年度读取思路，支持本地历史榜单目录导入。
- **安装体验完善**：全中文安装向导，支持自定义安装路径，自动创建目录。
- **抓取优先级修复**：优化榜单来源切换优先级，修复缓存回退逻辑导致的年份/月份不准确问题。

### v0.21 — 广告过滤
- **磁力内容校验（广告过滤）**：新增可选开关，读取磁力内部文件列表，判断是否混入 `.url`、`.apk`、`.html`、压缩包等广告/杂文件。
- **保守兜底**：读不到磁力元数据时不直接丢弃，回退到首条"未验证"候选，避免误伤正常结果。
- **与过滤词联动**：先执行原有"磁力过滤词"规则 → 再执行新增"磁力内容校验" → 最后返回最优磁力。

### v0.22 — 爬虫+整理一体化
- **爬虫 + 视频整理真正一体化**：新增"视频整理"工作区，同一软件完成爬取→导出→下载→整理→重命名→遗漏报告。
- **广告处理方式可视化**：整理配置中新增显式"广告处理方式"选择卡片（移入待删除 / 直接删除）。
- **全链路对账**：整理阶段对照爬虫番号名录，识别遗漏番号并输出补抓报告。
- **广告样本学习链路增强**：按番号自动学习，支持导入广告样本与正常样本，学习摘要实时刷新。
- **运行可视化与日志增强**：扫描进度、待整理进度、广告处理进度实时显示。
- **目录清理策略优化**：整理完成后自动清理空目录，避免目录结构越积越乱。

**0.22 同版补丁（2026-04-25）**：
1. 全中文修复：学习摘要、风险证据字段与状态提示中的英文残留全部改为中文。
2. 开头广告分流修复：命中开头广告风险的视频统一进入"含开头广告"。
3. 根目录收口修复：整理完成后根目录残留番号文件夹强制删除。
4. 完成态统计修复：任务完成文案新增"含开头广告"计数。

### v0.23 — AI 广告检测
- **内嵌 AI 广告检测模型**：新增三种可选模型策略：
  - `MobileNetV3 Lite（推荐）`：平衡速度与稳定性
  - `SqueezeNet Fast（更快）`：速度优先
  - `YOLOv8n Balanced（更严格）`：更敏感的风险判定
- **AI 广告检测可视化切换**：新增"是否启用 AI 广告检测"开关。
- **AI 模型类型选择**：客户可按设备性能和识别偏好切换模型策略。
- **主进程与检测服务全链路打通**：前端模型选择写入设置并传入整理流程。
- **学习摘要增强**：新增模型信息字段（activeModel / activeModelLabel）。
- **启动崩溃热修复**：修复 `DEFAULT_AD_MODEL_TYPE` 初始化顺序导致的启动崩溃问题。

### v0.24 — 日志优化（v0.23 补丁）
- **日志保留策略**：调整日志输出保留策略，确保关键信息不丢失。
- **开头广告粗筛精筛修复**：修复开头广告粗筛与精筛阶段的分流逻辑。
- **运行结果页优化**：优化运行结果页面的信息展示与交互。

### v0.25 — 整理优化（v0.23 补丁）
- **整理优先**：调整整理逻辑优先级，先完成番号识别和重命名，再做广告检测。
- **后置广告复核**：广告检测改为在改名/搬运后进行，命中转入"含开头广告"。
- **目录快速删除**：删除阶段引入"安全目录级快速删除"，优先整目录删除提升速度。
- **遗漏番号对账补抓**：学习链路与日志可视化增强。

### v0.26 — 大体积处理
- **大体积全入待整理**：大于等于最小体积的视频，无论是否命中爬虫番号，都先进入"待整理"。
- **未命中保留原名**：未命中爬虫番号或未识别番号时，不删除，保留原名，决策权交给客户。
- **删除速度优化**：视频移动到待整理文件夹后，原文件夹直接删除，不再逐文件扫描。
- **番号标准化修复**：
  - 修复 `normalizeFilmId` 函数，保守规范化策略（仅统一大写和分隔符，保留原始数字格式）
  - 解决特殊番号（如 `FNC-210548`）被错误格式化的问题
  - 从 JAV 网站 URL 提取的番号优先保留原始格式
- **女优榜单修复**：修复女优榜单默认显示第 36 名的问题（重置滚动位置）。
