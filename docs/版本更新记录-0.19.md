# JAV自动化爬虫工具 版本更新记录（截至 0.19）

更新时间：2026-04-12
当前正式版本：0.19

## 0.1
- 修复安装流程。
- 打通基础运行链路。
- 完成最初的桌面可用化准备。

## 0.2
- 桌面版 GUI 上线。
- 支持 Windows 直接打开使用。
- 从命令行工具正式迈向可视化桌面应用。

## 0.3
- 增加地址输入、输出位置、并行量、延迟等选项。
- 优化桌面布局与表单结构。
- 提升小白用户直接使用体验。

## 0.4
- 增强分页校验逻辑。
- 优化页锁定补抓。
- 完善尾页数量判断。

## 0.5
- 整体汉化桌面界面。
- 加入页数判断逻辑。
- 状态说明与主要交互统一中文化。

## 0.6
- 新增补爬功能。
- 增加弹窗提醒。
- 支持磁力 TXT 导出。
- 增加未完成番号显示。

## 0.7
- 修复日志乱码问题。
- 统一日志编码与显示逻辑。
- 提升问题排查体验。

## 0.8
- 加入任务状态落盘。
- 支持断点续爬与结果二次校验。
- 提升长任务中断后的恢复能力。

## 0.9
- 补充底层注释，方便后续维护与 AI 协作。
- 优化滚动区域与超长任务显示。
- 调整打包体积与资源组织方式。

## 0.10
- 优化解析容错。
- 调整终止流程。
- 强化补抓去重与缺失修复逻辑。

## 0.11
- 补充备用网址。
- 修复一批已知问题。
- 改善日志与页面卡顿相关问题。

## 0.12
- 大任务场景下增强稳定性。
- 增加磁力文档直接打开入口。
- 优化长时间运行过程中的操作体验。

## 0.13
- 补查分页缺口。
- 修正真实并发执行逻辑。
- 改为 HTTP 优先并引入批量写盘，减少 IO 开销。

## 0.14
- 修正完成判定逻辑。
- 启用索引页 / 快速 HTTP / Cloudflare 补抓三段队列。
- 提升大批量任务下的补抓效率。

## 0.15
- 大幅提升抓取速度。
- 增强抓取目标准确率与动态任务栏表现。
- 优化大任务抓取过程中的稳定性与速度平衡。

## 0.16
- 收口输出文件，仅保留核心结果文件与日志。
- 强化完成判定，自动生成未完成番号文件。
- 升级失败重试策略与分页可信度评分。

## 0.17
- 焕新桌面布局与状态面板，整体风格更年轻化。
- 补充版本历史展示，并修复 `limit` 模式下的超量入队问题。
- 强化中文显示、大任务稳态与缺口判断。

## 0.18
- 新增 FANZA 女优参考榜模块，支持月榜与年榜查看。
- 支持榜单点击后一键填充爬虫参数：自动定位真实女优目录、填入有磁力数量、每页条数与预计页数。
- 优化榜单区域界面：收窄筛选控件，约束超长网址显示，交互提示更直观。

## 0.19
- 继续推进桌面端与抓取核心的耦合拆分，降低后续改动的连带风险。
- 优化抓包速率与恢复链路，重点处理大任务下的 UI 卡顿与状态堆积。
- 新增背景自定义模块，支持用户自行更换桌面背景图片。
- 修复 base-url 快捷地址点击、日志面板与状态面板在高频更新场景下的卡顿问题。
